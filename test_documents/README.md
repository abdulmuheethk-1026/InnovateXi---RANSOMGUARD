# RANSOMGUARD – Attack Simulation Module

This module simulates early-stage ransomware behavior in a safe, non-destructive manner.

## Features
- Rapid file access simulation
- Bulk file renaming behavior
- Controlled test environment
- No encryption or data damage

## Purpose
To generate behavioral anomalies that can be detected early by a lightweight defense system.

## Disclaimer
This project is strictly for educational and defensive research purposes.
